Printf function

format: character string containing 3 types of objects :
	- plain characters (copied to standard output )
	- character escape sequences (must be converted and copied)
	- format specifications

variadic function: 
	- create a function that accepts multiple args
	- make a struct for the % characters 
	- make a function for each case of cspdiuxX%
		- %c Prints a single character.
		- %s Prints a string (as defined by the common C convention).
		- %p The void * pointer argument has to be printed in hexadecimal format.
		-%d Prints a decimal (base 10) number.
		- %i Prints an integer in base 10.
		- %u Prints an unsigned decimal (base 10) number.
		- %x Prints a number in hexadecimal (base 16) lowercase format.
		- %X Prints a number in hexadecimal (base 16) uppercase format.
		- %% Prints a percent sign
